daily units above schedule

Each bar is the insulin delivered above the configured basal schedule for that day. This is the primary signal for automated basal acting like correction insulin.

Period Basal Mix

scheduled vs extra

Shows how much basal matched the profile versus how much was additional automated delivery during the selected summary period.

Hourly Basal Rate Profile

average delivered vs configured

Top: configured basal profile compared with the average delivered basal rate for the selected period; the light band shows the observed delivered range. Bottom: average delivered minus configured by hour, so positive bars identify recurring basal correction windows.

Each daily bar is split by CGM range. Use this next to basal correction load to see whether extra basal coincided with more highs or lows.
